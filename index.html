FREELANCE CLIENT PROJECT

AutoReport Generation System

Full-Stack Development Internship Portfolio

The Mission: Automate a complex, manual motor insurance reporting workflow using generative AI.
Core Tech Stack: Python (Flask) | Google Gemini AI | JavaScript

🚀 90% Speed Improvement 📈 20% Client Capacity Increase

1. Multi-Modal PDF Ingestion

A secure Python Flask backend accepts complex, unstructured PDF documents containing Registration Certificates, Driving Licenses, Policies, Claim Forms, and Repair Invoices.

2. Gemini AI Extraction Engine

Leverages Google Generative AI (Gemini 2.5 Pro / 3 Flash) with multi-shot prompts to perform highly accurate optical character recognition and semantic extraction, normalizing non-standard insurance terminology into strict JSON payloads.

3. Dynamic Report Synthesis

Parsed JSON payloads are pipelined into Python's `FPDF` engine to dynamically generate Motor Final Survey Reports and Repair Assessment Summaries, including comprehensive part-level cost calculations and GST estimations.

4. Google Drive OAuth Sync

Integrates a fully authenticated Google OAuth2 flow, allowing the backend to autonomously push completed, sanitized PDF reports directly into the client's cloud storage over securely authenticated resumable upload sessions.
